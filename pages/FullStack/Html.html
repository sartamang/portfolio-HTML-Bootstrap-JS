HTML- Hyper Text Markup Language

HTML is the standard markup language for creating and structuring web pages. It utilises the sets of elements that instruct the browser on how to execute all the content (W3Schools, n.d). Every webpage relies on HTML alongside CSS and JavaScript. HTML is a static language, which means it does not provide interactive features on its own rather has to rely on CSS and JavaScript for styling and interactivity.

HTML Structure

This picture here shows the basic structure of HTML.

<!DOCTYPE html>
<html lang="en">
<head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
</head>
<body>

</body>
</html>


<!DOCTYPE html> => Defines the version of HTML

<html> => Root element of an HTML page

<head> => Contains information about the document

<title> => Sets the title of the page

<body> => Holds all the visible content

Many get confused with elements and tags. Although they are similar, HTML elements and tags are not the same thing. If there is content in a tag, then it becomes the element. Advantages

The advantages of HTML are given below:
  • Free and open
  • Simple and easy to learn
  • Lightweight
  • Supports multimedia
  • Widely supported

Limitation

HTML is a super-easy and powerful tool for structuring webpages; however, it also comes with some limitations.
  • Not a programming language, meaning it cannot perform logic.
  • Limited design control
  • No database
  • Security limitations
  • Static in nature

Conclusion

HTML is the foundation of the webpage as it gives a webpage its structure. Even though it cannot handle logic, styling or other things, without HTML, it will be hard to make a website. However, with the help of CSS and JavaScript, it becomes possible to create a full website with ease. Despite its limitations, HTML still remains the starting point for anyone learning web development or creating a website.
