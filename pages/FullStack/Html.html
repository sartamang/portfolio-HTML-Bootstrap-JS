HTML- Hyper Text Markup Language

HTML is the standard markup language for creating and structuring web pages. It utilises the sets of elements that instruct the browser on how to execute all the content (W3Schools, n.d). Every webpage relies on HTML alongside CSS and JavaScript. HTML is a static language, which means it does not provide interactive features on its own rather has to rely on CSS and JavaScript for styling and interactivity.

HTML Structure

This picture here shows the basic structure of HTML.

<!DOCTYPE html> => Defines the version of HTML

<html> => Root element of an HTML page

<head> => Contains information about the document

<title> => Sets the title of the page

<body> => Holds all the visible content

Tags and Elements

Many get confused with elements and tags. Although they are similar, HTML elements and tags are not the same thing. If there is content in a tag, then it becomes the element. While tag is the keyword that is surrounded in angle brackets (< >), and informs the browser what kind of content to expect (GeeksforGeeks, n.d.). Let’s look at some of the HTML tags:
  1. Heading tag

    2. There are a total of 6 headings. Each one has a different size, with h1 being the biggest one.

  2. Paragraph tag

    3. In a paragraph tag, we can write a paragraph or long text.
    The structure of this tag is < p > Paragraph < /p >

  3. Anchor tag

    4. An anchor tag is used to create a hyperlink in a webpage. This allows the user to navigate from one page to another just by clicking on the link.

  4. Listing tag

    5. HTML offers two types of listing, one is ordered and another is unordered.
    The tag structure of an ordered list => < ol > < /ol >
    The tag structure of an unordered list => < ul > < /ul >

     

    There is one more, and that is the description list. But it is not often used much.

  5. Table tag

    6. There is a tag named Table where we can create a table. All the data in the table should be inside this <table> </table> tag. Inside this, we use a different tag to insert the table data.

    < tr > < /tr > => This is a table row tag, it contains the data of one row.
    < th > < /th > => This is a table heading, this contains the heading of the table.
    < td > < /td > => this is table data, here all the data is inserted.

  6. Form tag

    7. Form tag is used to create a form. All the things are written inside the <form> </form> tag. Here are some of the tags that are used inside the form:

    <label> </label>=> This tag defines the label for an input element.

Advantages

The advantages of HTML are given below:
  • Free and open
  • Simple and easy to learn
  • Lightweight
  • Supports multimedia
  • Widely supported

Limitation

HTML is a super-easy and powerful tool for structuring webpages; however, it also comes with some limitations.
  • Not a programming language, meaning it cannot perform logic.
  • Limited design control
  • No database
  • Security limitations
  • Static in nature

Conclusion

HTML is the foundation of the webpage as it gives a webpage its structure. Even though it cannot handle logic, styling or other things, without HTML, it will be hard to make a website. However, with the help of CSS and JavaScript, it becomes possible to create a full website with ease. Despite its limitations, HTML still remains the starting point for anyone learning web development or creating a website.
