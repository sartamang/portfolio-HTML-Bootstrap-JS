CSS - Cascading Style Sheets

CSS is a language that tells the browser how things should be displayed in the browser screen or in data files. On its own, CSS cannot change the content written using HTML; however, CSS elevates the appearance of the content by making it eye-catching (W3Schools, 2025).

One thing to note about CSS is that it does not have a version, but rather has levels. Each new level is built on the previous one by refining its features and definition. Till now, there are 3 levels of CSS (World Wide Web Consortium, 2025).

CSS1 => 1996 AD, the first version.
CSS2 => 1998 AD added a new feature to the previous one.
CSS3 => 2004 AD - 2011 AD.

There is no CSS4 officially, but level 3 was updated and often referred to as CSS.

Core Concept

The core concepts are the principles that one needs to know before using CSS (GeeksforGeeks, 2025).
  1. Selectors

    2. We can select each div or an HTML element to apply the design.

    body{
      margin: 0px;
      padding: 0px
    }
    .container{
      background: #000;
      border-radius: 4px;
    }

  2. Properties

    3. CSS provides a lot of properties through which we can change the font sizes, colour and many other things.

    font-size: 20px;
    fonr-weight: bold;

  3. Unit

    4. Using the px, rem, em, vh, vw, and %, we can apply the length to a div or any particular HTML element.

    background-color: rgb(8,8,8);
    padding-left: 5px;
    padding-top: 4rem;
    box-shadow: 1em 1em 5em;

  4. Responsive design

    5. Media queries (@media) and flexible layouts (grid) are used to create the responsive design.

    @media (max-width: 768px){
      h1{
        color: green;
        font-size: 30px;
      }
    }

    /*for mobile max-width is 480px*/
    @media (max-width: 480px){
      h1{
        color: green;
        font-size: 20px;
      }
    }

  5. Transition and Animation

    6. Transition => To create a smooth change, esp. while applying the hover.

    img{
      transition: transform o.3s ease-in;
    }
    image:hover{
      transform: scale(1.5);
    }

    Animation => @keyframe is used to apply animations like bounce, rotate, and slide.

    h1{
      animation: bounce 1s infinite;
    }
    @keyframe bounce{
      0%, 50%, 100%{
        transform: translateY(0);
      }
      25%, 75%{
        transform: translateY(10%);
      }
    }

Types

There are in total 3 types of CSS through which we can apply it in HTML.

  1. Inline CSS

    2. In inline CSS, styles are applied directly to an individual HTML element using the <style> </style> attribute. In terms of hierarchy, inline CSS holds more importance, hence gets implemented 1st and even overrides the internal and external CSS.

    <h1 style= "font-size: 20px;"> Sunderland University </h1>

  2. Internal CSS

    3. Internal CSS is applied when the user/developer doesn’t want to make a separate file for CSS and wants to complete the project in one file. This is best for a small project. Usually, this is written outside the body tag.

    <style>
      h1{
        font-size: 20px;
      }
    </style>

  3. External CSS

    4. External CSS is mostly recommended by many professional developers. The main feature it provides is clarity. All the CSS properties and rules are written in seperate files, so, need to be connected using the <link> tag.

    ......
    <link rel = "stylesheet" href = "style.css">
    <body>

    </body>

Advantage

The advantages of CSS are:

  • Saves time
  • Offline browsing
  • Easy maintenance

Disadvantage

Despite providing all these designs to the HTML, there are still some limitations it has. They are:

  • Complexity in larger projects.
  • Limited dynamic feature.

Conslusion

CSS is used for styling webpages and files, but unlike programming languages, it does not include any logic, just as HTML. One must understand and learn the core concept initially in order to properly utilise CSS. Each of its types has its own advantages and use cases, where one type might be fast while another one might give us better maintainability. Without CSS, modern websites would not be any attractive as they are at present.
