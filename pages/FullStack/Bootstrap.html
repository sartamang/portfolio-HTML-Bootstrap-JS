Boot Strap

Bootstrap is an open-source front-end framework. Bootstrap is popular for creating responsive websites and web applications. Its pre-defined component, created with CSS and JavaScript/JQuery, makes it simpler for developers to design a website, and additional styles can be added through CSS. It also solves the issue that CSS have, which was browser compatibility (GeeksforGeeks, 2025).

The latest version of Bootstrap is Bootstrap 5, which was released on 16 June 2020.

Quick timeline on which version was released in which year.
2010-2011 => developed by Mark Otto and Jacob Thornton
2011           => public release
2012           => Bootstrap 2
2012           => Bootstrap 3
2018           => Bootstrap 4
2021           => Bootstrap 5

Features

Compared to the previous version, Bootstrap 5 have improved a lot, making it famous on the market again. The main features of Bootstrap 5 are: (GeeksforGeeks, 2024)
  • Uses Vanilla JavaScript
  • Responsive fonts
  • Icon library
  • No JQuery
  • Improved Grid system
  • Updated forms

How to use it

There are two ways through which we can use Bootstrap (Bootstrap, 2025).
  1. From CDN

    2. These links can be found on the homepage of the Bootstrap official page.

    • Bootstrap CSS Library

      • This link should be in the <head> tag.
      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-LN+7fdVzj6u52u30Kp6M/trliBMCMKTyK833zpbD+pXdCLuTusPj697FH4R/5mcr" crossorigin= "anonymous">

    • JavaScript Library

      • This script should be last after the footer inside the <body> tag.
      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/js/bootstrap.bundle.min.js" integrity="sha384-ndDqU0Gzau9qJ1lfW4pNLlhNTkCfHzAVBReH9diLvGRem5+R9g2FzA8ZGN954O5Q" crossorigin= "anonymous" ></script>

  2. Download

    3. Inside the official page of Bootstrap, click download. Unzip the folder, then link them to the HTML.

    inside the <head> tag
    <link rel= "stylesheet" href= "css/bootstrap.min.css" >
    at last inside the <body> tag
    < script src= "js/bootstrap.bundle.min.js" < </script>

Advantages

The advantages of Bootstrap are as follows:

  • Customising through CSS variables.
  • Quickly customise components.
  • Bootstrap icons.
  • Powerful JavaScript plugins without JQuery.

Limitations

Despite its huge improvement, there are still some limitations to it.

  • Heavy file size.
  • Similar designs.

Conclusion

Bootstrap is a powerful open-source collection of CSS and JavaScript. With each new version, it continued to improve the feature, maintaining a constant market pace for a long time. Its prebuild feature makes making a website much easier and saves the time of developer. Despite many advantages, there are some drawbacks. Nonetheless, it’s still one of the very well-known and powerful frontend frameworks.
